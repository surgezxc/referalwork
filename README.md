# Реферальный лендинг — практическая работа по HTML/CSS

**Демо:** https://surgezxc.github.io/referalwork/

## Структура страницы

Лендинг состоит из 5 основных блоков:

1. **Шапка (`<header>`)** — логотип, главный заголовок и подзаголовок с призывом к действию.
2. **Навигация (`<nav>`)** — якорные ссылки на разделы страницы: вакансии, заявка, вопросы.
3. **Основной контент (`<main>`)** — три секции:
   - **Вакансии (`<section id="vacancies">`)** — карточки вакансий в тегах `<article>`.
   - **Заявка (`<section id="application">`)** — форма обратной связи.
   - **FAQ (`<section id="faq">`)** — ответы на частые вопросы.
4. **Подвал (`<footer>`)** — копирайт.

## Использованные HTML-решения

| Элемент | Где используется |
|---|---|
| `<!DOCTYPE html>`, `<html lang="ru">` | Базовая структура документа |
| `<meta charset>`, `<meta viewport>` | Кодировка и адаптивность |
| `<title>`, `<meta description>` | SEO и заголовок вкладки |
| `<link rel="stylesheet">` | Подключение внешнего CSS |
| `<header>`, `<main>`, `<section>`, `<article>`, `<footer>`, `<nav>` | Семантическая вёрстка |
| `<h1>`–`<h3>` | Иерархия заголовков |
| `<p>`, `<strong>` | Текстовые блоки и выделение |
| `<ul>`, `<li>` | Списки требований к вакансиям |
| `<ol>`, `<li>` | Пошаговая инструкция в FAQ |
| `<img>` | Логотип в шапке и иконки компаний в карточках |
| `<a href>` | Навигация и кнопки «Устроиться» |
| `<form>`, `<label>`, `<input>`, `<select>`, `<textarea>`, `<button>` | Форма заявки |
| `<script>` | Подключение JavaScript для навигации и формы |

## Использованные CSS-решения

| Приём | Описание |
|---|---|
| **Селекторы** | По тегам (`body`, `h1`), классам (`.vacancy`, `.btn`), вложенности (`.vacancy .btn`) |
| **Цвета** | HEX (`#333`, `#4a6cf7`), `rgba()` для теней и прозрачности |
| **Градиенты** | `linear-gradient` для шапки и кнопок |
| **Отступы** | `margin`, `padding` для расстояния между блоками |
| **Шрифты** | `font-family: Arial, Tahoma`, `font-size`, `font-weight` |
| **Позиционирование** | `position: relative/absolute/sticky` для шапки, навигации и декора |
| **Псевдоэлементы** | `.vacancy::before`, `.faq::after` — декоративные полоски |
| **Анимации** | `@keyframes` для фоновых точек и вращения |
| **Интерактивность** | `:hover` на карточках и кнопках, `:focus` на полях формы |
| **Адаптивность** | `@media (max-width: 600px)` для мобильных экранов |
